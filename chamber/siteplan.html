Site Plan — Chamber of Commerce

Planning document for the WDD 231 “Chamber” site (Term 1, 2026).

1. Site Name

Name: Idaho Community Chamber

Reason: The name communicates a chamber that promotes local businesses and community opportunities across Idaho. It is simple, memorable, and clearly connected to economic/community support.

Optional domain idea: idaho-community-chamber.org

2. Site Purpose

The purpose of this website is to serve as a central hub for chamber information and community engagement. It provides membership details, a business directory sourced from a JSON data file, featured member spotlights, and a local weather section to support residents and visitors.

  • Explain chamber benefits and provide a way to join (Join page + form).
  • Display a member directory using data/members.json.
  • Show “Spotlight” members (random Silver/Gold members) on the home page.
  • Display current weather and a 3-day forecast from OpenWeather.

3. Scenarios

  • “Which local businesses offer web design, marketing, or IT services, and how can I contact them?”
  • “Who are the featured (spotlight) chamber members today?”
  • “What is the current temperature and the 3-day forecast for the area?”
  • “How do I join the chamber, and what membership level is best for my business?”

4. Color Scheme

The following colors will be used consistently across the site and are applied in this document.

Primary: #1F4E79

Used for: header background, headings, navigation, links, primary buttons.

Accent: #F4B400

Used for: highlights, section borders, badges, hover/active accents.

Background: #F7F9FC

Used for: page background and spacing areas.

6. Typography

  • Poppins — used for headings (H1–H3) to create a strong hierarchy.
  • Inter — used for body text for readability across devices.

These fonts are already applied in this site plan document.

Project Content Alignment (Members, Spotlights, Weather)

Member Directory Data (data/members.json)

The directory is populated from a JSON file containing member objects with the following fields:

  • name, address, phone, website, image, tagline
  • membership numeric levels: 1 = Bronze, 2 = Silver, 3 = Gold

Current members included:

  • Idaho Websites (Gold) — Web design & marketing
  • Surge Web Design (Silver) — Modern sites for business
  • Graphic Zen (Bronze) — Branding & creative
  • Peppershock Media (Silver) — Digital marketing
  • Flash Tech Services (Bronze) — IT services
  • Metro Express Car Wash (Gold) — Car wash membership
  • Boise Coffee Co. (Bronze) — Coffee & community

Spotlights (scripts/spotlights.js)

The spotlight section selects only Silver (2) and Gold (3) members, shuffles them, and randomly displays 2 or 3 spotlight cards on the home page.

Eligible spotlight members from the current JSON:

  • Gold: Idaho Websites, Metro Express Car Wash
  • Silver: Surge Web Design, Peppershock Media

Each spotlight card displays: member name, logo image, membership label, phone, address, and a website link.

Weather (scripts/weather.js)

The weather section uses the OpenWeather forecast endpoint (5-day / 3-hour data). It displays:

  • Current temperature (using the first forecast item)
  • Current description (capitalized words)
  • 3-day forecast (filters midday entries: 12:00:00 and shows the next three days)

Note: The current implementation uses imperial units (Fahrenheit) and fixed coordinates in the script.
